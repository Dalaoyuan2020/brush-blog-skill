# OPEN_SOURCE.md — 开源发布说明

## 📢 仓库已公开（2026-02-23）

本仓库已从 **私有** 改为 **公开**，正式开源发布 v1.0。

## 🎯 为什么公开？

1. **M8 阶段已完成**，核心功能稳定可用
2. 让社区用户可以下载体验、提 Issue、贡献代码
3. 开源项目本来就是边开发边公开的

## 📌 版本管理策略

- 使用 `git tag` 标记稳定版本
- `v1.0` = M8 完成，首个公开版本
- 后续开发（M9-M12）继续在 `main` 分支进行
- 每完成一个里程碑，打一个新 tag（v1.1, v1.2...）

```bash
# 查看所有版本
git tag -l

# 用户下载稳定版
git checkout v1.0
```

## ⚠️ 给 Codex 的说明

**这个库现在是公开的！** 开发时请注意：

1. **不要提交敏感信息**（API Key、Token、密码等）
2. **不要提交用户数据**（profiles/、behavior_events.jsonl、content.db）
3. `.gitignore` 已配置好，会自动排除这些文件
4. **提交信息要清晰**，因为所有人都能看到
5. **代码质量要注意**，公开库代表项目形象

### 开发流程不变

```
Codex 开发 → push 到 main → 小羊一号 VPS 测试 → 打 tag 发布
```
